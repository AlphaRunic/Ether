Fundamentals

Ether is a high-level dynamically-typed interpreter. It has C-style syntax.
Some basic fundamental concepts in Ether include:

print "Hello, world!";

##:
  Globals and locals are the same thing when defined in the global scope.
  When a local variable is defined inside of a nested scope, only that scope may access that variable's value.
  When a global variable is defined inside of a nested scope, any scope may access that variable's value.
:##
global a = "outer";
{
  local a = "inner";
  print a;
}
print a;

local foo = "bar";
for (local i = 0; i < 10; i++)
  foo = foo + " bar"; ## I plan to implement compound assignment, but it's not implemented yet.

local count = 0;
while (true) {
  count = count + 1;
  if (count == 10)
    break;
}

if (count != 10)
  raise "Something went very wrong.";
else
  print "Everything is normal.";

method sayHello(name) {
  print "Hello, " + name + ".";
}

sayHello("world");
sayHello("Jane");
